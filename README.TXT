PROJECT TITLE: UNIHELP
PURPOSE OF PROJECT: El propósito de este proyecto es proporcionar una plataforma donde los estudiantes universitarios puedan encontrar tutores calificados para reforzar sus conocimientos en diversas materias.
VERSION or DATE: Version 2.0 - Diciembre 2024
HOW TO START THIS PROJECT:Instalar Java 8 o superior en el sistema y posteriormente ejecutar el instalador, finalmente abrir el archivo Unihelp.jar

Tecnologías usadas en el desarrollo de la aplicación:
Lenguaje de programación: Java
Entorno de desarrollo: BlueJ
Gestión de dependencias: Uso de Gson para serialización/deserialización de JSON
Interfaz gráfica: Swing (Java GUI)

Tecnologías necesarias para que el proyecto funcione:
Java Runtime Environment (JRE) 8 o superior
Sistema operativo compatible con Java (Windows, macOS, Linux)

USER INSTRUCTIONS:
Abra la aplicación ejecutando el archivo Unihelp.jar.
- Si quiere concoer más de la aplicación seleccione la opción de "Acerca de la aplicación"
- Si desea salir de la aplicación seleccione la opción "Salir"
- Para entrar al catalogo de tutores seleccione la opción "Ver lista de tutores", 
una vez ahí puede seleccionar al tutor de su preferencia, 
la aplicación cuenta con un filtro de tutores según la materia en la que se especializan, 
una vez se seleccione al tutor de su preferencia, comuniquese con el en el número de contacto que se muestra en pantalla, 
una vez concretado un encuentro seleccione la opción de match para registrar el encuentro en la aplicación, 
si no se concretqa ningún encuentro seleccione la opción volver.
